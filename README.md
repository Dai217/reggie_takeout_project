# reggie_takeout_project
本项目是参考B站黑马程序员的《瑞吉外卖》教程所实现的一个SpringBoot+MybatisPlus+Mysql技术栈的前后端分离外卖管理系统。

技术栈：

SpringBoot 
MybatisPlus 
Mysql 
Session 

后台 :
 登录模块 
 员工管理 
 分类管理 
 菜品管理 
 套餐管理 
 订单管理 
 
前台:
 用户模块 
 购物车模块 
 地址模块 
 订单模块 
 菜品模块 
访问后台：localhost:8080/backend/page/login.html即可 
访问前台：localhost:8080/front/page/login.html
后续改建：加入redis
